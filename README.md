## Installation

```bash
git clone git@github.com:LemuelGuevara/hyprland-dotfiles.git ~/hyprland-dotfiles
```

Stow afterwards to apply symlinks:

```bash
cd ~/hyprland-dotfiles
stow --adopt --target=$HOME/.config .
```
